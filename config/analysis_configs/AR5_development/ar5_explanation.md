AR-5 Developmental Trajectory example variant

This example variant points at the processed child gaze fixations file and filters for participant_type == 'infant'.
Use this variant as a template to create cohort-specific variants mirroring other ARs.

Fields:
- variant_key: unique key used to name result subdirectory
- cohorts: list of cohort definitions (key, label, data_path, participant_filters)
- output: export options
